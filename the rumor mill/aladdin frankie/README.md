<h2>The Frankie virus for Aladdin emulators</h2>

This virus sounds deeply weird. It apparently only spreads to Macintosh applications *under the Aladdin Macintosh emulator*.
On a native Macintosh it does not spread. It needs the emulator.

There are credible writeups; but no sample found.<br><br>

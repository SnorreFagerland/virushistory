False Alarm 1.00

False Alarm sets an alarm in the Time application each time the
3a/3c/Siena is turned on.

Copy SYS$TIM.OPO to any directory on the target machine.

Highlight the RunOPL icon, press Tab, navigate to SYS$TIM.OPO and press
Enter.

WARNING: removing False Alarm will involve a soft reset or use
of tools such as Spy to kill the SYS$PRGO process.

README for VIRUS
----------------

   Okay, I know what your're thinking - who would download a virus on purpose?<P>
   Well, the thing is, is that this isn't really a virus.  It won't hide on your machine and delete everything when you least expect it.  It is just a little joke, which could probably be used to scare those who are less computer literate into thinking that they have a virus.  It consists of one OPO file.

   Run it, and it will return you to the system screen as if nothing has happened.  Everything works normally for a few seconds.  Then, up pops a little window saying that something is being transferred (presumably by infra red).  You can still use the pen to switch between apps and to open files, and in fact all the normal things you can do with the pen, but whenever you switch apps, this upload window "follows" you.

   When it has "downloaded", a warning message flashes up.  Then a small black square starts making its way around your screen.  The pen is still usuable, but this square prevents the keyboard from working properly!

   So there you go, that is it.  Download it and try it.  Its just a bit of fun, and you will probably delete it, or send it to a friend via infra red to give them a shock! :-)

   There is of course one other thing...

****  PLEASE READ THIS BEFORE YOU RUN VIRUS ****
   To get rid of the square :

        - Press Return<BR>
        - Type in "leave me alone" (ensure Caps lock is off)<BR>
        - Press Return<BR>

   and it will close the program.

   Oh, and one final thing.  I accept no responsibility for this program.  It is well behaved, and I have had no trouble with it.

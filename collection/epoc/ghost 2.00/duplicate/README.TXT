Ghost In The Machine Version 2.00

Ghost In The Machine sends insulting messages in the form of infoprints
to running built-in applications on the Series 3a/3c.

Messages are more frequently sent to the system screen.

See the screenshot on my site for an example.

No expletives are contained in these messages but I can build special
versions on demand :>

Copy SYS$INS.OPO to any directory on the target machine.

Highlight the RunOPL icon, press Tab, navigate to SYS$INS.OPO and press
Enter.

Uses asynchronous I/O and the TIM: device.

WARNING: removing Ghost In The Machine will involve a soft reset or use
of tools such as Spy to kill the SYS$PRGO process.
